Debemos ser configurables

Es genial tratar de que todos los elementos que estén sujetos a cambiar dentro de nuestra función se mantengan como parámetros configurables: nombres de clases, literales, colores...

Si no usamos ningún framework podríamos usar una función que nos permitiese combinar valores de dos objetos distintos, por ejemplo:


    var merge = function (o, ob) {
      var z;
      for (z in ob) {
        if (ob.hasOwnProperty(z)) {
          o[z] = ob[z];
        }
      }
      return o;
    }

De esta manera podríamos preparar nuestra clase:


myClass = function (options) {
  if(!(this instanceof myClass)) return new myClass(options);
  this.options = merge({
    option1: "I'm the option one",
    option2: "I'm the option two"
  }, options || {});
  
  ...
  
}

Así a la hora de instanciarla podríamos usar los valores por defecto o personalizarlos si nos interesa.


var obj = new myClass();
  => option1: "I'm the option one"
  => option2: "I'm the option two"

var obj = new myClass({option1: "oh!", extra: "oh!"});
  => option1: "oh!"
  => option2: "I'm the option two"
  => extra:   "oh!"
